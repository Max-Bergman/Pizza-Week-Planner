# Pizza-Week-Planner

Vite + React app for planning Portland Pizza Week routes.

## Deploy on Vercel (free)

1. Sign in at [vercel.com](https://vercel.com) with GitHub.
2. **Add New… → Project** and import **Max-Bergman/Pizza-Week-Planner**.
3. Leave defaults: **Framework Preset** Vite, **Build Command** `npm run build`, **Output Directory** `dist`.
4. Click **Deploy**. Future pushes to `main` deploy automatically.

`vercel.json` includes SPA rewrites so deep links keep working if you add client-side routes later.

## Local

```bash
npm install
npm run dev
```
